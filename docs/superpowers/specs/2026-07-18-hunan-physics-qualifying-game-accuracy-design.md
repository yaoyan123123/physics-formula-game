# 粉笔黑板大作战：湖南高中物理合格考准确性升级设计

日期：2026-07-18  
状态：用户已确认设计方向，待实施计划

## 1. 目标

在保留现有粉笔黑板视觉风格和公式记忆核心玩法的前提下，把游戏校准为“湖南省普通高中物理学业水平合格性考试”基础训练工具：

- 知识范围只覆盖《普通高中课程标准（2017 年版 2020 年修订）》规定的物理必修内容；
- 难度止于合格性考试，不引入选择性考试或高考压轴式综合推导；
- 修正公式、定义、章节、适用条件、方向、正负号和单位中的不准确或过度绝对化表述；
- 让挑战模式借鉴湖南物理合格考客观题结构，但明确它不是完整试卷，也不预测官方是否合格；
- 增加自动化内容校验，降低以后修改时重新引入知识错误的风险。

## 2. 权威依据与产品边界

### 2.1 最新范围与考试时量

湖南省教育考试院 2026 年通知规定：各科考试内容为教育部《普通高中课程方案和语文等学科课程标准（2017 年版 2020 年修订）》规定的必修课程内容；物理考试时量为 60 分钟。来源：

- [湖南省教育考试院：2026 年普通高中学业水平合格性考试统考具体事项](https://jyt.hunan.gov.cn/jyt/sjyt/hnsjyksy/web/ksyzcfg/202603/t20260311_33931704.html)

### 2.2 物理试卷公开结构

湖南省教育考试院公开的物理合格性考试试卷结构为：

- 单项选择题 18 题，每题 3 分，共 54 分；
- 实验题 3 题，每题 4 分，共 12 分；
- 计算题 3 题，共 34 分；
- 全卷 60 分钟、100 分。

来源：

- [湖南省教育考试院：湖南省普通高中学业水平合格性考试试卷结构](https://jyt.hunan.gov.cn/jyt/sjyt/hnsjyksy/web/ksyzcfg/202203/t20220304_22739429.html)

本游戏只模拟上述“18 道单项选择题”的基础知识训练体验，不模拟实验题和书面计算题的作答、步骤评分或整卷时长。

### 2.3 合格判定边界

《湖南省普通高中学业水平考试实施办法（2019 年修订）》规定，合格标准依据基本教学质量要求以及当次命题和考试情况确定，不应把游戏中的固定百分比宣传为湖南官方合格线。来源：

- [湖南省教育厅：普通高中学业水平考试实施办法（2019 年修订）](https://jyt.hunan.gov.cn/jyt/sjyt/xxgk/tzgg/201912/t20191231_994028.html)

因此游戏删除“60 分即官方合格”和 A—E 等级，不提供官方成绩预测，只提供训练反馈。

## 3. 当前版本主要问题

### 3.1 章节结构

当前必修三后两章映射错误：

- 第 12 章应为“电能 能量守恒定律”；
- 第 13 章应为“电磁感应与电磁波初步”；
- “能源”不是独立的第 13 章，应回到第 12 章相关内容中处理。

### 3.2 必修内容缺项

当前公式库没有完整覆盖闭合电路欧姆定律及电源电动势、内阻等必修基础内容，却使用“全 13 章核心公式”的强表述。公式总数“67”还是硬编码文案，容易在补充内容后失真。

### 3.3 知识表述不严谨

已确认需要修正的典型问题包括：

| 当前表述或问题 | 修订原则 |
| --- | --- |
| 动能“恒为正” | 改为“动能为非负标量，可以为 0” |
| 效率“永远小于 100%” | 改为定义上不超过 100%；实际装置通常小于 100% |
| `F = kx` 无条件 | 标明弹性限度内，并区分弹力大小与方向 |
| `P = Fv` 无条件 | 同向时才可写成 `Fv`；一般形式说明夹角 |
| 库仑力、点电荷场强把带符号电荷直接当大小 | 大小使用绝对值，方向另行说明 |
| `U = W/q` 缺端点 | 使用 `U_AB = φ_A - φ_B = W_AB/q` 并说明正负约定 |
| `F = BIL` 无条件 | 标明导线与磁场垂直；一般情况含 `sinθ` |
| 平行板电容公式符号体系含混 | 使用 SI 口径 `C = ε₀εᵣS/d` |
| `I = q/t` 表述过粗 | 使用 `Δq/Δt`，说明平均或恒定电流情境 |
| 欧姆定律笼统写成适用于金属和电解液 | 改为一定条件下的欧姆导体，不作材料类别的绝对判断 |
| 机械能守恒、摩擦做功表述绝对化 | 明确研究系统和外力/非保守力做功条件 |
| 第一宇宙速度结论无条件 | 限定近地圆轨道、忽略空气阻力和地球自转等理想条件 |
| 超重失重公式无测量情境 | 限定竖直方向支持力或拉力模型，并强调重力不变 |

内容审查不只修改公式本身，还同步修改公式名、知识卡说明、易错点和概念题答案，防止同一知识点在不同位置出现互相矛盾的口径。

## 4. 内容模型设计

公式卡继续保存在静态页面的数据区，避免为本轮升级引入后端或构建系统。每条内容至少具备以下语义字段：

- 唯一标识；
- 教材章节与必修册；
- 公式或核心关系；
- 符号和常用单位；
- 适用条件；
- 方向、正负或参考系说明；
- 一条合格考层级易错提醒；
- 可生成的题型标签。

页面展示保持“先看公式、再揭示条件”的记忆节奏。适用条件不会藏在只有教师才会发现的长段落中，而是作为公式卡的固定区域显示。

公式总数由数据实时计算，界面不再手写“67 条”。补齐必修内容后，允许总数自然变化，不为维持营销数字删减必要知识。

## 5. 湖南合格考挑战设计

### 5.1 名称与免责声明

模式名称改为：

> 湖南学考·18 题公式冲刺

入口附近明确显示：

> 对应湖南物理合格考单选题数量的专项训练，不是完整试卷，不预测官方合格结果。

### 5.2 题目结构

每轮固定 18 道单项选择题，全部只有一个最佳答案：

- 6 题：公式辨认、物理量含义与单位；
- 4 题：适用条件、方向、正负号和易错表述；
- 6 题：代入一步或常见两步基础计算；
- 2 题：图像、实验原理或生活情境中的基础判断。

三册必修内容每轮都要出现，题目按知识主题分层抽取，避免完全随机造成整册或大主题缺席。13 个章节不要求每轮全部出现，但通过跨轮轮换提高长期覆盖率。

原有判断题不再以“对/错”形式混入所谓模拟卷；需要保留的概念判断改写为四选一单项选择题。

### 5.3 难度上限

允许：

- 直接公式识别；
- 单位换算；
- 单对象、单过程的一步计算；
- 常见两步计算；
- 教材典型图像、实验原理和生活情境；
- 对适用条件和典型错误的辨析。

不允许：

- 多对象多过程耦合；
- 复杂临界与分类讨论；
- 高考压轴式天体、传送带、板块、电磁综合；
- 超出必修要求的定量电磁感应推导；
- 依赖复杂数学技巧才能完成的计算。

### 5.4 计时与结果

根据题型设置合理的游戏计时：识别题较短，条件题和基础计算题适当延长。超时只影响本轮游戏，不宣传为真实考试能力结论。

结果页显示：

- 答对 `x/18`；
- 正确率；
- 三册必修的薄弱主题；
- “继续巩固 / 基本掌握 / 较为稳固”等训练反馈。

不显示“官方合格/不合格”、固定 60 分合格线或 A—E 官方等级。

## 6. 界面与交互

- 保留黑板、粉笔、擦除反馈、连击等现有美术和核心交互；
- 首页和挑战页统一写明“湖南高中物理合格考·必修范围”；
- 公式卡新增简洁的“条件”和“易错”粉笔标签；
- 增加可展开的“湖南卷说明”，展示 60 分钟、100 分及 18+3+3 的完整结构；
- 不新增外部图片素材，本轮不改变视觉方向；
- 手机端不得因新增说明造成按钮拥挤或滚动区域不可用。

## 7. 学习日期修复

当前连续学习日期使用 UTC 日期，湖南本地时间凌晨至早上可能被计入错误日期。改为按 `Asia/Shanghai` 本地自然日生成日期键，并增加跨午夜测试。

## 8. 自动化验证

保持静态站点和单页部署方式，新增 Node 测试覆盖：

1. 13 章名称和顺序准确；
2. 第 12、13 章映射准确；
3. 必修关键主题存在，包括闭合电路欧姆定律；
4. 公式总数由数据计算，不再残留硬编码 67；
5. 已知错误措辞不再出现；
6. 条件型公式必须带适用条件；
7. 18 题挑战始终为单项选择，正确答案唯一；
8. 每轮覆盖必修一、二、三；
9. 难度标签只允许湖南合格考层级；
10. 学习日期按中国本地自然日计算；
11. 主源文件和部署版 `index.html` 保持一致；
12. 现有答案预览测试继续通过。

人工验收补充检查：桌面和手机布局、键盘操作、触摸操作、计时反馈、公式渲染、结果页用语及随机抽题的长期覆盖。

## 9. 非目标

本轮不做：

- 完整 60 分钟湖南合格考仿真卷；
- 实验题在线操作和计算题步骤评分；
- 高考选择性考试内容；
- 用户账号、云端错题本、排行榜或教师后台；
- 全面重做视觉风格；
- 未经用户明确授权的线上发布。

## 10. 完成标准

满足以下条件才可视为本地升级完成：

- 公式与概念内容完成逐条审查；
- 湖南范围、章节和挑战文案准确；
- 18 题模式符合单项选择专项训练定位；
- 所有新增和既有测试通过；
- 桌面与手机关键流程验收通过；
- 主源文件与部署副本一致；
- 未把训练结果宣传成湖南官方合格结论。
